# Contributing

Contributions should keep the repository reproducible, well-tested, and easy to adopt.

Before opening a change:

- use Node `20.x` via `.nvmrc`
- run `npm ci`
- run `npm run lint`
- run `npm test`

General expectations:

- keep changes focused and intentional
- update documentation when public behavior changes
- add or update tests when contract behavior, SDK behavior, or example agreements change
