# Public P0 evidence

`p0-linea-sepolia-package-deployment.json` is generated by
`npm run proof:p0:linea`. It binds the canonical package digest, EIP-712 permit,
signed CREATE2 salt, predicted clone, deployment transaction, deployed runtime,
and on-chain agreement fields to the RPC-verified protocol deployment.

Anyone can recompile the checked-in package and verify the receipt from public
RPC state without a private key:

```sh
npm run check:p0:linea
```
